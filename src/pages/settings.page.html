Settings Core

The Core adapter provides the base functionality that all other adapters rely on. Core can not be disabled. (Maintainer: Kendraio)

 DDEX RIN

DDEX RIN is the "Recording Information Notification" data standard. This adapter enables support for import and export of RIN data, as well as the core entities for storage of projects, people, recordings, and works. See the DDEX website for more information. (Maintainer: Kendraio)

RIN can not be disabled in the demo app.
